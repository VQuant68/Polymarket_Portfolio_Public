1   Installation

1.1   Prerequisites

SoftwareVersionNotes
Node.js≥ 18.xJavaScript runtime
npm≥ 9.xPackage manager
TypeScript≥ 5.6Included in devDependencies

1.2   Install Dependencies

cd Solana_bot
npm install

Installs all dependencies: @solana/web3.js, @solana/spl-token, @triton-one/yellowstone-grpc, axios, borsh, bs58, dotenv, tweetnacl, and dev tools.

1.3   Environment Configuration

Step 1: Create .env from template

cp .env.example .env

Step 2: Fill in credentials

MASTER_PRIVATE_KEY=    # base58 or JSON array [1,2,3,...]
RPC_URL=               # e.g. https://devnet.helius-rpc.com/?api-key=xxx
JITO_ENDPOINT=         # e.g. https://mainnet.block-engine.jito.wtf
GRPC_URL=              # e.g. https://grpc.helius.xyz
PINATA_API_KEY=        # from pinata.cloud (free tier)
PINATA_SECRET_KEY=     # from pinata.cloud
VariableDescriptionRequired
MASTER_PRIVATE_KEYMaster wallet private key (base58 or JSON array)Always
RPC_URLSolana RPC endpoint (Helius, QuickNode, etc.)Always
JITO_ENDPOINTJito Block Engine URLFor bundles
GRPC_URLYellowstone gRPC endpointFor streaming
PINATA_API_KEYPinata IPFS API keyFor launch
PINATA_SECRET_KEYPinata IPFS secret keyFor launch
⚠️ Security: NEVER commit the .env file to Git. It is already excluded via .gitignore.

2   Running Commands

2.1   Offline Tests (no credentials needed)

npm run test:offline

Runs 40 pure logic tests — discriminators, instruction building, PDA derivation, bonding curve math, wallet I/O, Jito tip. No .env or internet required.

2.2   Devnet Tests (requires RPC_URL)

npm run test:devnet

Runs all 48 tests (40 offline + 8 devnet) — RPC Pool, Blockhash Cache, balance check, and nonce funding on live Solana Devnet.

2.3   Full Integration Test

npm run test:full

2.4   Production Build

npm run build          # Output: dist/

2.5   Same-Block Buy-Sell (standalone tool)

npx ts-node src/tools/same-block-buy-sell.ts

Configure tokenMint inside the file before running. Uses Jupiter Quote API + Jito bundle.
